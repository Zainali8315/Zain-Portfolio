# Zain Ul Abideen — Portfolio Website

A responsive, animated static portfolio website with:

- About, skills, education and contact sections with scroll-reveal animations
- Animated hero entrance, rotating "currently exploring" typewriter line, scrolling tech ticker
- Animated stat counters, hover micro-interactions on cards/links/buttons, active-section nav highlighting, scroll progress bar
- RentalX and AI Resume Screening Agent project cards
- Direct project and GitHub links
- Dedicated resume page with a **Text Resume / Image View toggle** — real selectable, ATS-friendly text content plus the original designed image, side by side
- Resume download and browser Print/Save PDF options
- Mobile navigation and fully responsive layout
- Respects `prefers-reduced-motion` for accessibility

## Files

- `index.html` — main site
- `styles.css` — all styling and animations
- `script.js` — mobile nav, scroll-reveal, counters, typewriter effect, scroll progress, active nav state
- `resume.html` — resume page (text + image toggle)
- `assets/` — put your `profile.jpg` and `resume.jpg` here (not included in this package)
